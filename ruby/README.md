# Advent Of Code - Ruby

## Usage

```
rake aoc:auth[cookie]         # Set Cookie for your AOC Account Copied from the browser
rake aoc:benchmark[year,day]  # Benchmark Solution
rake aoc:help                 # help
rake aoc:init[year,day]       # fetch templates
rake aoc:run[year,day]        # Run Solution

* year format = YYYY
* day  format = DD
```
